BIM 3D

Over the last few years, BIM (Building Information Modeling) has permeated the Architecture, Engineering and Construction (AEC) industry thanks to BIM software like Revit, ArchiCAD and Tekla.

BIM models made with these applications can be seen as CAD models (2D and 3D graphical representations) plus extensive meta data, like planning data, cost data and life-cycle data.

With BIM 3D by Enuma Technologies, BIM models can be presented in an intuitive and state-of-the-art manner to an audience outside the building team, like facility managers, property tenants and the general public, via webbrowsers, mobile devices or Virtual Reality headsets.

Approach

Approach

A typical approach to translate BIM models to BIM 3D involves the following steps:

  • Convert the original BIM model to BIM 3D format using EnumaVR Export tools
  • Define the target audience for the BIM model (ex. facility managers, tenants, general public, etc.)
  • Define data filters and data access level for each target group (what data must be shown, what data must be editable/read-only, etc)
  • (Optional) Connect external data repositories to the BIM 3D model (ex. include data from existing facility management systems)
  • Develop the user-interface for each target group for interacting with the BIM 3D information (ex. website, mobile app, VR app)
  • Publish

EnumaVR Export tools

Enuma Technolgies has developed various export tools to convert BIM models and other 3D models to BIM 3D. Emphasis is placed on data optimalization and reduction, without losing essential information.

Data reduction, especially for 3D geometrical data, is essential in order to support various hardware, like consumer laptops/PCs and mobile devices.

Data Filters

Data filters can be applied to BIM data in order to present only necessary information to the target audience.

Data filters can also be used to rename and translate BIM data to recognizable terminology for the target audience.

View Filters

View filters can be applied on the BIM 3D model to show and hide 3D objects based on various criteria, like discipline, due maintenance, etc.

View filters can be freely set by the user, or 'hard set', depending on the target audience.

Role Filters

Role filters are a powerful way to filter BIM data for a specific target audience, using the above mentioned data and view filters.

Additionally, role filters can be used to present BIM data and external data in a role-specific user-interface.
